# POO — Herencia y Polimorfismo en Java

Proyecto orientado a practicar los principios de **herencia**, **polimorfismo**, **sobrecarga** y **sobrescritura** mediante un sistema de empleados y representantes de ventas.

## Características
- Clase base `Employee` con atributos generales.
- Subclases:
  - `SalesRep`: ventas individuales.
  - `SalesManager`: gestiona equipos de representantes.
- Polimorfismo aplicado en cálculo de ventas y métodos sobrescritos.
- Manejo de colecciones con `HashMap` para equipos.

## Estructura
src/classes/
├─ Employee.java
├─ SalesRep.java
├─ SalesManager.java
├─ User.java
└─ Main.java

## Tecnologías
- Java 8+
- Eclipse IDE

## Ejecución
Ejecuta `Main.java` para probar creación de empleados, polimorfismo y cálculos de ventas.
